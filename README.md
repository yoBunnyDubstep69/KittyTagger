# KittyTagger
A GUI program that simplifies adding geolocation data to the image.

Supports .png .jpg and .jpeg types of image. Built-in english translation. You can check geolocation using YandexMaps on Russian version and GoogleMaps on English version.

HOW TO USE:

Adding Geotag:
1)Go to YandexMaps or GoogleMaps and copy coordinates of the chosen place.
2)Press "Add Geotag" and agree on using clipboard data or type coordinates manually.
3)Choose the image
4)Done

Removing Geotag:
1)Choose the image
2)Done

Viewing geotag
1)Choose the image
2)Watch your geotag on GoogleMaps

# KittyTagger
Программа с графическим интерфейсом, которая упрощает добавление данных геолокации к изображению.

Поддерживает типы изображений .png .jpg и .jpeg. Встроенный перевод на английский язык. Вы можете проверить геолокацию с помощью ЯндексКарты на русском языке и GoogleMaps на английском.

КАК ИСПОЛЬЗОВАТЬ:

Добавить геотег:
1)Зайдите на ЯндексКарты или GoogleMaps и скопируйте координаты выбранного места.
2)Нажмите "Добавить геотег" и одобрите использование данных из буфера обмена или введите координаты вручную.
3)Выберите изображение
4)Готово

Удаление геотега:
1)Выберите изображение
2)Готово
